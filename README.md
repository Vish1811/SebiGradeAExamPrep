## 📖 Exam Overview

| Phase | Description | Weightage |
|------|------------|----------|
| 🥇 Phase I | Screening Test | Qualifying |
| 🥈 Phase II | Merit-Based | 85% |
| 🥉 Interview | Final Selection | 15% |

---

## 🎯 Key Focus Areas

### Phase I
- Programming (Java/C++)
- SQL & DBMS
- Networking
- Cyber Security

### Phase II (🔥 Most Important)
- Data Structures (40%)
- Algorithms (30%)
- OOP (20%)
- Strings (10%)

---

## 🧠 Preparation Strategy

- 🎯 Focus **70–80% effort on Phase II**
- 🧪 Practice **code tracing + debugging**
- 📊 Weekly mock tests
- 🔁 Multiple revisions of high-weight topics
